# **App Name**: Zekkers

## Core Features:

- KYC Verification: Implement KYC (Know Your Customer) verification process for employers and candidates, ensuring trust and security within the job portal.
- AI-Assisted Matching: Use AI to analyze job postings and candidate profiles, providing accurate and relevant job matches with explainability for users.
- Job Posting & Management: Enable employers to post jobs with a flexible form, manage job listings through various lifecycle stages (draft, published, expired, archived), and track views and applications.
- User Authentication and Profiles: Offer multiple authentication methods (email/password, Phone OTP, Google, LinkedIn, GitHub) and progressive profile builders with verified badges.
- AI Resume Parsing Tool: Automatically parse resumes to populate job seeker profiles. Integrate third-party resume parsing API or Cloud Function ML model.
- Referral Tracking and Rewards: Implement referral code generation and tracking for employers and campus partners. Attribute hires to specific referrals.
- Admin & Moderation: Provide an admin console for user management, content moderation, and site settings.

## Style Guidelines:

- Primary color: Indigo (#3740FF) for a modern, trustworthy feel.
- Accent color: Teal (#06B6D4) for highlights and CTAs.
- Background color: Light Gray (#F5F5F7) for a clean, accessible interface.
- Font pairing: 'Poppins' (sans-serif) for headers and short text chunks, and 'PT Sans' (sans-serif) for body text, for readability.
- Code Font: Use 'Source Code Pro' for displaying code snippets within the platform, particularly in developer-related job descriptions or profiles.
- Use modern, clear icons with subtle animations for a delightful user experience.
- Incorporate subtle animations like parallax scrolling, card hover elevation, and skeleton loaders for smooth transitions and user engagement.
